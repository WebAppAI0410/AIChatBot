# AI アシスタントプロジェクト理解用ドキュメント

このフォルダには、AIアシスタント（Claude）がAIChatBotプロジェクトを理解し効率的に編集するために作成したドキュメントが含まれています。

## ドキュメント一覧

1. **project_structure.yaml**
   - プロジェクトのディレクトリ構造と各ファイルの役割を俯瞰
   - 主要コンポーネントとモジュールの把握に利用

2. **data_flow.md**
   - アプリケーションのデータフローを図解
   - 各機能の処理の流れと依存関係を理解するのに使用

3. **feature_file_mapping.md**
   - 機能別に関連するファイルをマッピング
   - 特定の機能を修正する際に参照するファイル群を特定

4. **modification_plan_summary.md**
   - プロジェクトの修正計画の概要
   - 実装済み項目と今後の実装予定を整理

5. **tech_stack.md**
   - プロジェクトで使用している技術スタックの概要
   - 使用ライブラリやフレームワークの全体像把握

## 使用方法

- 新しい機能実装や修正を行う前に、これらのドキュメントを参照して全体像を把握
- 該当機能の関連ファイルを特定し、データフローを確認して影響範囲を把握
- 修正計画の順序と依存関係を考慮して作業を進める
- 実装中に発見した新しい情報や変更点があれば、これらのドキュメントを更新

## 注意点

- これらのドキュメントはAIアシスタントの理解を助けるためのもので、正式なプロジェクトドキュメントではありません
- 実際のコードと矛盾がある場合は、常にコードを信頼してください
- プロジェクトの進行に伴い、適宜このドキュメントを更新していきます
