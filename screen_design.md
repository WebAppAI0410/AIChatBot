## 2. ModelSelect 詳細 UI
```
Auto (最適なモデルを自動選択)   ▸
Qwen3:4B (ローカル)   [⚪️ 未DL]  ▸
```
- Auto: プラン別に最適なモデルを自動選択（現在使用中のモデル名を小さく表示）
- バッジ: ⚪️未DL / 🔄DL中 / 🔍検証中 / 🟢使用可 / 🔴エラー
- 未DLタップ → InstallLocalModel
