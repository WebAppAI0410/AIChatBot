```

Qwen3:4B (ローカル)   [⚪️ 未DL]  ▸
```
- Auto: プラン別に最適なモデルを自動選択（現在使用中のモデル名を小さく表示）
- バッジ: ⚪️未DL / 🔄DL中 / 🔍検証中 / 🟢使用可 / 🔴エラー

# ノート機能 画面設計（最新版）

## ノート一覧画面
```
┌────────────────────────────┐
│ [←] ノート        ✨        │ ← 右上AIアシスト
│                            │
│ [📁 フォルダA]              │
│ [📝 ノート1]                │
│ ...                        │
│                            │
│        ＋                  │ ← 右下FAB（タブと重ならない位置）
└────────────────────────────┘
```
- FABはタブUIと重ならないよう絶対配置
- 右上「✨」でAIアシスト（ノート全体管理・一括タグ編集）

## ノートエディタ画面
```
┌────────────────────────────┐
│ [←] ノート編集    ✨        │ ← 右上AIアシスト
│                            │
│ [タイトル]                  │
│                            │
│ [WYSIWYGエディタ]           │
│  （タップで編集モード）      │
│                            │
│ タグ: #AI #メモ             │
└────────────────────────────┘
```
- タイトル・本文ともタップで編集モード、入力は自動保存
- 保存ボタンは廃止

### テキスト選択時のアクションバブル
```
[WYSIWYGエディタ]
  ・・・ノート本文・・・
  [選択範囲] ← 長押し or ダブルタップで選択
        └─ [AIアシスト] ← アクションバブル
```
- 本文内テキスト長押しまたはダブルタップで部分選択
- 選択時に「AIアシスト」アクションバブルが表示される
- タップで部分編集AIアシストドロワーが表示

### AIアシストドロワー（ノート/一覧共通）
```
┌────────────────────────────┐
│ AIアシスト  [質問/編集/部分] │ ← モード切替
│                            │
│ [チャットUI]                │
│                            │
│ [プロンプト入力欄]          │
│ [送信]                      │
└────────────────────────────┘
```
- 「質問」モード：ノート内容に関するQ&Aのみ、AIは編集不可
- 「編集」モード：AIがノート全体を編集提案、差分確認・適用可能
- 「部分」モード：選択テキスト部分のみを編集（テキスト選択時のみ表示）
- ノート一覧AIアシスト：ノート全体管理や複数ノートのタグ一括編集なども可能

### 部分編集AIアシストドロワー
```
┌────────────────────────────┐
│ AIアシスト  [部分編集]      │
│                            │
│ 選択部分:                   │
│「・・・選択テキスト・・・」   │
│                            │
│ [プロンプト入力欄]          │
│ [送信]                      │
│                            │
│ AI提案:                     │
│「・・・整形済みテキスト・・・」│
│ [この部分に適用] [キャンセル] │
└────────────────────────────┘
```
- テキスト選択からアクションバブル経由で呼び出し
- 選択部分のみにAIの編集・整形提案を適用可能
- ドロワー表示中は背面編集不可（閉じると編集再開可能）
